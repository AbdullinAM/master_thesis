
\keywords{%

}

\abstractcontent{
  Данная работа посвящена исследованию возможностей объединения абстрактной
интерпретации и метода ограниченной проверки моделей с целью повышения 
производительности анализа. Исследование было проведено на основе системы 
Borealis, которая использует метод ограниченной проверки моделей для анализа 
программ. Borealis работает на основе системы LLVM, поэтому были 
проанализированы существующие инструменты абстрактной интерпретации для системы 
LLVM. На основе системы Borealis был разработан модуль абстрактной интерпретации 
для LLVM IR. Полученный прототип был апробирован на наборе реальных программных 
проектов. Апробация показала, что объединение абстрактной интерпретации и метода 
ограниченной проверки моделей повышает точность и производительность анализа.
\\
}

\abstractcontenten{
  This work is dedicated to analysis of methods of abstract interpretation
and bounded model checking. The technique for combining AI and BMC was proposed. 
The main goal of this technique is to improve the efficiency of static analysis.
The work is based on bounded model checker Borealis, which uses LLVM and Clang 
for code analysis and modification. Analysis of existing frameworks for abstract interpretation of LLVM IR was 
performed. Abstract interpretation framework prototype for LLVM IR was developed.
Results of conducted experiments show that proposed technique allows to improve
precision and efficiency of analysis.
}
