\abstractcontent{
Магистерская работа посвящена исследованию возможностей объединения абстрактной
интерпретации и метода ограниченной проверки моделей. Исследование было
проведено на основе системы Borealis, которая использует метод ограниченной
проверки моделей для анализа программ. Borealis работает на основе системы LLVM,
поэтому были проанализированы существующие фреймворки абстрактной интерпретации
для системы LLVM. Был разработан свой фреймворк абстрактной интерпретации для
LLVM. Данный фреймворк был внедрен в систему Borealis. Полученный прототип был
апробирован на наборе реальных програмных проектов. Апробация показала, что
объединение абстрактной интерпретации и метода ограниченной проверки моделей
повышает качество и производительность(??) анализа
}
